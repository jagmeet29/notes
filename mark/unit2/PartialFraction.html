Partial Fraction

Partial fraction decomposition is a technique used to break down a complex rational function into simpler fractions. This is particularly useful in calculus for integration. The basic idea is to express a given rational function as a sum of simpler fractions.

For example, consider the rational function:

$$ \frac{2x + 3}{(x - 1)(x^2 + 1)} $$

We can decompose it as follows:

$$ \frac{2x + 3}{(x - 1)(x^2 + 1)} = \frac{A}{x - 1} + \frac{Bx + C}{x^2 + 1} $$

where A, B, and C are constants to be determined.

Ways to Solve Partial Fractions

Partial fraction decomposition is a technique used to break down complex rational expressions into simpler fractions. Here are the main methods to solve partial fractions:

Standard Decomposition Method

This is the most common approach and follows these steps:

  1. Ensure proper rational expression: The degree of numerator must be less than the degree of denominator. If not, perform polynomial long division first.
  2. Factor the denominator: Break it down into:
    • Linear factors (x - a)
    • Repeated linear factors (x - a)ⁿ
    • Irreducible quadratic factors (ax² + bx + c)
  3. Set up partial fractions: Write one fraction for each factor:
    • For linear factor (x - a): A/(x - a)
    • For repeated linear factor (x - a)ⁿ: A₁/(x - a) + A₂/(x - a)² + ... + Aₙ/(x - a)ⁿ
    • For irreducible quadratic factor (ax² + bx + c): (Bx + C)/(ax² + bx + c)
  4. Eliminate fractions: Multiply all terms by the least common denominator (LCD).
  5. Find coefficients: Several methods exist for this step (explained below).

Methods for Finding Coefficients

1. Substitution Method

After multiplying by the LCD, substitute values that make factors of the denominator zero. For example, if (x - 2) is a factor, substitute x = 2.

2. Equating Coefficients Method

After multiplying both sides by the LCD:

  • Expand the expressions
  • Collect like terms
  • Create equations by matching coefficients of same powers of x
  • Solve the resulting system of equations

3. Residue Method

This method uses complex analysis:

  • Calculate the residue of the rational function at each pole
  • For simple poles, the residue can be calculated using: Res(f, a) = lim(x→a) [(x-a)f(x)]
  • These residues directly give the coefficients in the partial fraction decomposition

4. Limit Method

Multiply both sides of the equation by a particular factor from the denominator and take the limit as x approaches the root of that factor. This isolates one coefficient by eliminating other terms.

Special Cases

  • Improper Fractions: When the degree of numerator ≥ degree of denominator, perform polynomial long division first.
  • Irreducible Quadratic Factors: Over real numbers, quadratics with negative discriminants can't be further factored. Use the form (Ax + B)/(ax² + bx + c).
  • Complex Roots: If working in complex number field, all polynomials can be factored into linear terms, simplifying the decomposition.
