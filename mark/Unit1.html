Unit 1

Propositional Logic

Any random statement that is either true or false.

Logical Connectives

  • Negation: Opposite

    P ¬P
    True False
    False True

  • Conjunction: And

    P Q P ∧ Q
    True True True
    True False False
    False True False
    False False False

  • Disjunction: Or

    P Q P ∨ Q
    True True True
    True False True
    False True True
    False False False

  • Exclusive Or: Either or

    P Q P ⊕ Q
    True True False
    True False True
    False True True
    False False False

  • Implication: If...then

    P Q P → Q
    True True True
    True False False
    False True True
    False False True

  • Bi Conditional: If and only if

    P Q P ↔ Q
    True True True
    True False False
    False True False
    False False True

Others

  • Inverse: (¬p → ¬q)

  • Converse: (q → p)
  • Contrapositive: (¬q → ¬p)
  • Equivalence: (p ↔ q)
