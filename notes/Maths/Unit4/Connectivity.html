Connectivity and Path

Important Terms

Walk: Alternative sequence of vertices and edges
Open Walk: A Walk is said to be open if its initial ans terminating vertices are different.
Closed Wal: A Walk is said to closed if its initial and terminating vertices are same.
Path: An open walk in which no vertex can appear more than once is called path.
Trial: An open walk is said to be trial if the edges are not repeated but the vertices may be repeating.
Simple Path: A path which does not contain the same edges more than once (no repeating edges).
Circuit: It is a walk in which no edges repeat.

Compliment of a Graph:

The complement of a graph G, often denoted as G̅ (G with a line over it) or G' is a graph that:
  • Has exactly the same set of vertices as the original graph G
  • Contains precisely those edges that are NOT present in the original graph G
In simpler terms, if two vertices are connected by an edge in the original graph, they will NOT be connected in the complement graph. Conversely, if two vertices are NOT connected in the original graph, they WILL be connected in the complement graph

Dijkstra’s algorithm

Dijkstra’s algorithm is a procedure for finding the shortest paths from a single source vertex to all other vertices in a weighted graph with non‐negative edge weights

Steps of Dijkstra's Algorithm

  1. Create a table with all vertices of the graph as columns (a, b, c, d, e, f, g, z).
  2. Initialize the first row with the starting vertex (a) as 0 and all other vertices as ∞.
  3. Choose the vertex with the minimum distance value (initially a with distance 0).
  4. For the selected vertex, update the distances to all its adjacent vertices if the new calculated distance is less than the current recorded distance.
  5. Mark each updated distance with the path in parentheses.
  6. Mark the selected vertex as visited (shown with a bold value in the table).
  7. Select the unvisited vertex with the minimum distance as the next vertex to process.
  8. Repeat steps 4-6 until all vertices have been visited.
